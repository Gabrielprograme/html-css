background-repeat-x

O atributo diz respeito a repetição da imagem na horizontal.

exemplo:

        
            

    body{

   background-image: url('https://github.com/gustavoguanabara/html-css
   /blob/master/imagens/mascote.png? raw=true');
   background-repeat: no-repeat;
   /* background-repeat-y and x */ 
   background-repeat: repeat-x; 

}

background-repeat-y

o atributo diz respeito a repetição da imagem na vertical
